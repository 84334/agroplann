# Welcome to the AgroPlan project

# Title: Smart Crop Rotation & Farming Assistant

## Project Overview
**AgroPlan** is a smart crop planning platform designed to help small-scale and beginner farmers make better planting decisions.
Many farmers lack knowledge of proper crop rotation and crop selection, which can lead to reduced soil nutrients, lower yields, and decreased income. In addition, there is no simple and accessible tool to help farmers plan their next crops effectively.
**AgroPlan** addresses these challenges by providing easy-to-understand guidance on crop selection and crop rotation. This helps improve soil health, increase productivity, and support sustainable farming practices, ultimately contributing to better food security and farmer livelihoods in Malaysia.

## Installation / Setup Guide
No installation is required.
Users can directly access the platform through a web browser on desktop or mobile devices.

## Technologies Used
**Frontend:** React + TypeScript + Tailwind CSS
**Backend:** Lovable Cloud (Database, Auth, Edge Functions)
**AI:** Google Gemini 3 Flash — powers the farming chatbot
**Weather:** Open-Meteo API (free, real-time)
**Payments:** Stripe (freemium model)
